# Розробка власного логотипу
## Мета

Навчитись створювати власну векторну графіку та використовувати її у 3D-моделюванні (для гравіювання або видавлювання).

## Завдання

Розробити власний логотип у векторному редакторі Inkscape та використати його у 3D-моделі в Autodesk Fusion 360.

## Вимоги до логотипу

- простий, читабельний дизайн
- мінімум 1 замкнутий контур
- відсутність дрібних деталей (< 0.5 мм)
- товщина ліній ≥ 0.8 мм
- формат експорту: SVG


## Вимоги до 3D-моделі
- форма: довільна (прямокутник, овал, фігурна)
- товщина основи: 2–3 мм
- логотип: об’ємний (виступ або гравіювання)
- висота або глибина логотипу: 0.8–2 мм
  
## Етапи виконання
1. Розробка логотипу (у Inkscape)
   - створити ескіз
   - використати прості форми (кола, лінії, текст)
   - додати текст
   - перетворити текст у криві (Path → Object to Path)
   - перевірити замкнутість контурів
2. Експорт
   - зберегти файл у форматі .SVG
3. Імпорт у Autodesk Fusion 360
   - Insert → Insert SVG
   - вибрати площину
   - масштабувати
4. Інтеграція в модель. Виконати:
   - Extrude (видавлювання) або
   - Cut (вирізання)

## Критерії оцінювання
- якість логотипу (простота, читабельність)
- коректність SVG
- правильний імпорт
- акуратність інтеграції в модель
- придатність до друку
