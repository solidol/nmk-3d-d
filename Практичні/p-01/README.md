# Створення брелка з текстом
## Мета роботи

Навчитись створювати просту 3D-модель з використанням тексту та базових операцій.

## Завдання

Створити 3D-модель брелока з власним ім’ям або словом (3–10 символів) та підготувати її до 3D-друку.


## Вимоги до моделі
- форма: довільна (прямокутник, овал, фігурна)
- товщина основи: 3–5 мм
- текст: об’ємний (виступ або гравіювання)
- висота тексту: 0.8–2 мм
- отвір для кільця:
    - діаметр: 4–6 мм
    - відступ від краю: ≥ 3 мм

## Етапи виконання
1. Створити ескіз основи
2. Виконати операцію видавлювання
3. Додати ескіз тексту
4. Виконати операцію видавлювання
5. Додати векторний малюнок (за бажанням)
6. Виконати операцію видавлювання
7. Додати ескіз отвіру
8. Виконати операцію видавлювання (вирізання)
9. Додати фаски (скругління) до отвору
10. Експортувати STL
11. Підготувати модель у слайсері

## Критерії оцінювання
- правильність геометрії
- читабельність тексту
- наявність малюнку
- коректний отвір
- якість друку
