# Брелок у вигляді шахової фігури
## Мета роботи

Опанувати операцію обертання (Revolve) та створення симетричних форм.

## Завдання

Створити брелок у формі шахової фігури (пішак, тура, ферзь тощо) з отвором для кріплення.

## Вимоги до моделі

- використати операцію Revolve
- висота фігури: 40–70 мм
- основа повинна бути стійкою
- отвір для кільця (4–6 мм)
- модель має бути цілісною (без окремих частин)

## Етапи роботи

1. Намалювати профіль фігури
2. Виконати Revolve
3. Додати отвір
4. Додати власний логотип
5. Перевірити товщини
6. Підготувати до друку


## Оцінювання
- правильність форми
- симетрія
- акуратність моделі
